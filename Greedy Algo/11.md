alid Parenthesis String

Given a string s containing '(', ')', and '\*', return true if it's valid.

Rules:
'(' must have a matching ')'.
')' must have a preceding '('.
'\*' can be '(', ')', or '' (empty).

Examples:

Input: s = "()"  
Output: true

Input: s = "(\*)"  
Output: true

Input: s = "(\*))"  
Output: true
